# RpsAI by SWE-1.5

## Основные принципы:

Анализ паттернов поведения - алгоритм собирает статистику по каждому игроку индивидуально
Адаптивное предсказание - использует несколько методов для предсказания следующего хода
Автоматическая очистка памяти - удаляет статистику старше 2 часов

### Методы анализа:

1. Частотный анализ
   Считает как часто каждый игрок выбирает камень, ножницы или бумагу
   Использует это как базовую вероятность для предсказания
2. Анализ последовательностей (паттернов)
   Ищет пары последовательных ходов (например, "камень-бумага", "ножницы-камень")
   Если игрок часто играет определенную последовательность, предсказывает следующий ход в паттерне
3. Анти-циклический анализ

Люди избегают повторять один и тот же ход много раз подряд
Если игрок 3 раза подряд сделал одинаковый ход, вероятность что он продолжит снижается

Работа с памятью:
Map<string, PlayerStats> - хранит статистику по логину игрока
Автоматическая очистка - каждый час удаляет данные старше 2 часов
Эффективное использование - только реальные ходы (без скрытых 'h')
